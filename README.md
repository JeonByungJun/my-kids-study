# 어린이 공부방

아이들을 위한 재미있는 학습 웹앱입니다.

## 주요 기능

- **받아올림 덧셈** — 두 자리 수 덧셈 (받아올림 문제만 출제)
- **받아내림 뺄셈** — 두 자리 수 뺄셈 (받아내림 문제만 출제)
- **캔버스 풀이** — 손가락/펜으로 자유롭게 풀이 과정 작성
- **숫자패드 입력** — 큰 버튼으로 정답 입력
- **PWA 지원** — 홈 화면에 추가하여 앱처럼 사용, 오프라인 지원

## 기술 스택

- React 19 + TypeScript
- Vite
- react-router-dom
- vite-plugin-pwa
- Vercel 배포

## 시작하기

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 배포

GitHub에 push하면 Vercel에서 자동 배포됩니다.

## 페이지 구조

```
메인 (과목 선택)
└── 수학
    ├── 연산
    │   ├── 받아올림 덧셈
    │   └── 받아내림 뺄셈
    ├── 도형 (준비 중)
    └── 통계 (준비 중)
└── 과학 (준비 중)

```
